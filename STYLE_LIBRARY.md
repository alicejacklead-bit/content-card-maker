# Style Preset Library

## Purpose

This library defines reusable visual style presets for `content-card-maker`.

It does not decide which style to use.

Style selection is owned by `VISUAL_ENGINE.md`.

This file only provides the structured definition of each preset so the
selected style can be applied consistently across cover, content cards,
and ending cards.

---

## Usage Rule

When `VISUAL_ENGINE.md` selects a style preset:

1. Use the matching preset in this file.
2. Keep one preset as the dominant visual system for the whole package.
3. Allow minor variation between pages, but do not change the core
   background logic, color logic, typography direction, or illustration
   direction.

---

## Preset Format

Each preset uses the same fields:

- Description
- Best Use Cases
- Visual Feeling
- Background
- Color
- Typography
- Icon Style
- Illustration Style
- Composition
- Spacing
- Things to Avoid
